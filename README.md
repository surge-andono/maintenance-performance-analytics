# Maintenance Performance Analytics

## Overview
This project delivers an end-to-end maintenance analytics solution, covering:
- Data Warehouse (Star Schema)
- Exploratory Data Analysis (EDA)
- KPI & Measure Design
- Power BI Dashboards
- Executive, Operational, and RCA decision support

## Analytics Flow
Raw Data → EDA → KPI Definition → Dashboards → Decisions

## Dashboards
1. Executive Overview  
   Focus: Strategic risk, cost, and performance direction

2. Maintenance & Operations  
   Focus: Workload, execution, and efficiency

3. RCA & Risk Analysis  
   Focus: Root cause diagnosis and corrective action planning

## Repository Structure
- documentation: Design document & storytelling
- diagrams: EDA flow and data warehouse diagrams
- eda: EDA presentations and mapping
- powerbi: Power BI dashboard file
- data_sample: Sample dataset for demonstration

## Tools
- Power BI
- DAX
- Star Schema Data Warehouse
